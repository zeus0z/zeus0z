### Fullstack RAG AI Platform

End-to-end AI Retrieval-Augmented Generation (RAG) system built with a NestJS (Node.js) backend and React frontend, featuring vector indexing (FAISS), semantic search, Redis caching, Azure Vector Database integration, and context-controlled LLM responses.

**Technical Highlights**

- Retrieval-Augmented Generation pipeline with vector similarity search
- Hybrid vector strategy using FAISS and Azure Vector Database
- Context window control for deterministic LLM responses
- Redis caching layer for retrieval optimization
- Modular backend architecture structured in feature-based modules
- Clear separation between retrieval, embedding, and generation layers
- Frontend interface for controlled prompt submission and response visualization
